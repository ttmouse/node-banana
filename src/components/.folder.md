# Folder: src/components/

> Trigger: Update when responsibilities or file list meaningfully change.
> 触发：当职责或关键文件列表发生"实质变化"时更新。

React UI components for workflow editor. / 工作流编辑器的React UI组件。
Downstream of store/; consumed by app/page.tsx. / store/ 的下游；被 app/page.tsx 消费。
All components are client components ("use client"). / 所有组件均为客户端组件。

## Core Files
- `WorkflowCanvas.tsx`: Main React Flow canvas / 主画布容器
- `FloatingActionBar.tsx`: Node creation toolbar / 节点创建工具栏
- `Header.tsx`: App header with workflow controls / 应用头部与工作流控制

## Subdirectories
- `nodes/`: Node type components (ImageInput, Prompt, NanoBanana, LLMGenerate, etc.)
- `edges/`: Edge type components (EditableEdge, ReferenceEdge)
