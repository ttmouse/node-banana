# Folder: src/

> Trigger: Update when responsibilities or file list meaningfully change.
> 触发：当职责或关键文件列表发生"实质变化"时更新。

Application core: UI components, state management, API routes. / 应用核心：UI组件、状态管理、API路由。
Downstream of app/, services consumed by client. / 被 app/ 消费的下游服务层。
All files must be client-safe (no secrets) except api/ routes. / 除 api/ 路由外，所有文件需客户端安全。

## Subdirectories
- `app/`: Next.js App Router (pages, API routes, layout)
- `components/`: React UI components (nodes, edges, modals)
- `store/`: Zustand state management
- `utils/`: Utility functions (image processing, caching)
- `types/`: TypeScript type definitions
- `constants/`: Application constants
